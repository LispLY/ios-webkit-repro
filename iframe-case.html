How to use: open in Edge or Chrome for iOS on an iOS 26.5 device. The subframe below tries the trigger automatically on load; if the popup was blocked (no user gesture), tap the button inside the frame. Reproduced = the tab crashes to "Can't open this page" (sad tab). Not reproduced = the frame keeps logging and the tab stays alive.

Structure: the trigger runs inside an <iframe> (a subframe). The frame calls window.open('about:blank') and closes it immediately. The popup content is irrelevant — no PDF, redirect, or SSO is needed.
